HL7 FHIR API

Overview

SyntheticMass exposes all synthetic medical record data using an HL7 FHIR (Fast Healthcare Interoperability Resources) v.1.8.0 compatible server. FHIR is a standard for exchanging healthcare data electronically and takes a modern, web-based approach to API design. For information about how to use FHIR, please visit the HL7 FHIR v.1.8.0 Documentation. SyntheticMass uses the open source GoFHIR FHIR server.

SyntheticMass FHIR Endpoint: https://syntheticmass.mitre.org/fhir/

Examples

The below examples demonstrate types of queries that can be executed against the SyntheticMass FHIR server using your browser. This is not an exhaustive list of the query capabilities of FHIR or the GoFHIR server implementation. Please review the HL7 FHIR v.1.8.0 Documentation for more information.

Basic Examples

  • All patients, paginated at 100 records:
    https://syntheticmass.mitre.org/fhir/Patient?_count=100
  • Basic information about a single patient:
    https://syntheticmass.mitre.org/fhir/Patient/58b3663e3425def0f0f6a229
  • All medical record data related to a patient:
    https://syntheticmass.mitre.org/fhir/Patient/58b3663e3425def0f0f6a229/$everything

Search Examples

  • Patients between the ages of 18 and 70:
    https://syntheticmass.mitre.org/fhir/Patient?birthdate=gt1947-03-26&birthdate=lte1999-03-26&_count=100
  • Patients with sinusitus:
    https://syntheticmass.mitre.org/fhir/Patient?_has:Condition:subject:code=444814009,75498004,36971009,40055000
  • Peak flow meter results for a patient:
    https://syntheticmass.mitre.org/fhir/Observation?code=http://loinc.org|19926-5&patient=58b3663e3425def0f0f696a9
  • Patients that have had a nasal endoscopy procedure:
    https://syntheticmass.mitre.org/fhir/Patient?_has:Procedure:patient:code=112790001
